# Product — Intent

## What It Is

Intent is a developer workspace for agent orchestration. It's what comes after the IDE.

## Core Features

### Agent Orchestration
Run multiple agents in parallel without conflicts. Agents delegate relevant context to each other without the developer pasting context into prompts.

### Developer Workspace
Agents, terminal, diffs, browser previews, and git operations in one self-contained workspace. Every workspace is an isolated git worktree.

### Living Spec
A spec that evolves as agents work. Developers define what, agents handle how. Agents read from and write back to the spec, staying aligned as code changes.

## Additional Features

- Full git workflow: prompt to commit to PR to merged without leaving the app
- BYOA (Bring Your Own Agent): works with Claude Code, Codex, OpenCode

## Real Claims (safe to use in ads)

- Multiple agents, one workspace
- Agents share context automatically
- No conflicts between parallel agents
- Isolated git worktrees per workspace
- Spec-driven: define what, not how
